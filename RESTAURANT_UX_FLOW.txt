## Design Philosophy

1. PERSONAL BENEFIT FIRST
   - "12 safe items" not "Help community"
   - Value signal, not altruism

2. INTELLIGENCE REDUCES FRICTION
   - GPS suggestions > Manual search
   - Cache = Instant repeat scans
   - Rank by relevance

3. NON-BLOCKING
   - Bottom sheet (not modal)
   - See menu photo behind
   - Natural gestures

4. SMART SKIP FRICTION
   - Small text link (not button)
   - Confirmation required
   - Loss framing message
   - Still allows skip

5. ADAPTIVE COUNT
   - High confidence (near) = 3 suggestions
   - Lower confidence (far) = 5 suggestions
